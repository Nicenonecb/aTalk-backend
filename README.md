

## aTalk

### Introduction
aTalk is an application that assists in language learning through AI.

### Features
[x] Refactoring by OpenAI 
[x] Support multiple languages
[x] Email Verification
[x] User Profile
[X] Replace key API


### Run
```bash
go run main.go
```

### Build
```bash
go build
```

#### Requirements
go 1.20 or above
